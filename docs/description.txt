Databench is a data analysis tool using 
<a href="http://flask.pocoo.org/">Flask</a>, 
<a href="http://socket.io/">Socket.IO</a> and 
<a href="http://d3js.org/">d3.js</a> with optional parallelization with 
<a href="http://python-rq.org/">Redis Queue</a> and visualization with 
<a href="http://mpld3.github.io/">mpld3</a>. 
Check out the <a href="http://databench-examples-viewer.svenkreiss.com/">live demos</a>.
