To contribute to Databench, fork the GitHub repository and clone it to your local machine. Then install your local version in editable mode into a virtualenv:

# inside an empty directory:
git clone https://github.com/<username>/databench.git .

virtualenv venv
source venv/bin/activate
pip install -e .

Now you can run

databench

and open http://localhost:5000 in a web browser. Now you are ready to submit pull requests of your contributions. Don’t forget to add yourself to AUTHORS.rst.
